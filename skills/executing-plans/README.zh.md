# 执行计划

## 概述

加载计划，批判性评审，执行所有任务，完成后报告。

**开始时声明：**“我正在使用 executing-plans 技能来实现该计划。”

**注意：** 告知人类伙伴，Superpowers 在有子代理支持的平台上效果更佳（如 Claude Code 或 Codex）。如有子代理，优先用 superpowers:subagent-driven-development。

## 流程

### 步骤1：加载并评审计划
1. 阅读计划文件
2. 批判性评审——发现疑问或担忧
3. 有疑问：在开始前与人类伙伴沟通
4. 无疑问：创建 TodoWrite 并继续

### 步骤2：执行任务

对每个任务：
1. 标记为 in_progress
2. 严格按步骤执行（计划已拆分为小步骤）
3. 按要求运行验证
4. 标记为 completed

### 步骤3：完成开发

所有任务完成并验证后：
- 声明：“我正在使用 finishing-a-development-branch 技能完成本次工作。”
- **必须调用子技能：** superpowers:finishing-a-development-branch
- 按该技能验证测试、展示选项、执行选择

## 何时停止并寻求帮助

**遇到以下情况立即停止执行：**
- 阻塞（缺依赖、测试失败、指令不清）
- 计划有关键缺口无法启动
- 不理解指令
- 验证多次失败

**遇到不确定时优先澄清，不要猜测。**

## 何时回溯前序步骤

如发现计划有重大遗漏或错误，应及时回到前序步骤修正。
